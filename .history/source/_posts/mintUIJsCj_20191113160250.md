## 下载插件

`npm i mint-ui -s`

### 全局导入使用--main.js 中

```javascript
import MintUI from "mint-ui";
import "mint-ui/lib/style.css";
Vue.use(MintUI);
```

**然后就可以直接使用 css 的封装组件了，如果要使用 js 的组件的话，需要按需加载需要的组件，官方文档都有**
