## 下载

``` js
npm i mint-ui -s
```

## 使用

### 全局导入使用--main.js中

``` js
import MintUI from 'mint-ui'
import 'mint-ui/lib/style.css'
Vue.use(MintUI)
```

**然后就可以直接使用css的封装组件了，如果要使用js的组件的话，需要按需加载需要的组件，官方文档都有**
