## 安装

``` js
npm i moment -s
```

## 引入

``` js
import moment from 'moment';
```

## 配置

``` js
Vue.filter('dateFormat',function (dataStr,pattern='YYYY-MM-DD HH:mm:ss') {  
  return moment(dataStr).format(pattern)
})
```
