## 修改打包指令和路径：

### 在webpack3中 是使用
``` js
webpack  .\src\main.js  .\dist\bundle.js 
```

### 在webpack4中分离出了webpack-cli （所以要先下载webpack-cli）来进行以后的操作;

``` js
npm i webpack-cli -g (全局下载)
webpack .\src\main.js -o .\dist\bundls.js -- mode development //进行打包
```
