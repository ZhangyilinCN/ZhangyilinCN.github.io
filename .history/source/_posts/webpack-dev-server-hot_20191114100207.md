## 下载

**需要安装 webpack-dev-server**

``` js
npm i webpack-dev-server -d //安装到本地依赖(只需要装到本地项目中，不需要-g)
```

**因为webpack-dev-server是依赖于本地的webpack和webpack-cli的，所以，如果本地中没有安装。则需要先安装，才能使用**

``` js
npm i webpack -d
npm i webpack-cli -d
```
