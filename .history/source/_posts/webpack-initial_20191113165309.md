## 使用

```js
npm i webpack -g //进行全局安装（本地的话使用后缀 -d）
```

## 安装之后再项目中使用

```js
npm init -y  //生成package.json文件
```

## 如果要在项目中装插件什么的 可以使用

```js
npm i jquery -s //（-s是项目中要使用的，-d是项目实际运行时不需要的，如gulp）

```
